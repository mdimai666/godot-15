# 15 Puzzle Game 🧩

A classic 15-puzzle sliding tile game built with **Godot 4.x** and **.NET (C#)**. Designed for mobile platforms (Android/iOS) with touch controls, smooth animations, and modern UI.

## ✨ Features

- 🎮 **Classic 15-Puzzle Gameplay** – Slide tiles to solve the puzzle
- 📱 **Mobile-First Design** – Touch-optimized controls and responsive UI
- 🔄 **Shuffle Algorithm** – Guaranteed solvable puzzles every time
- 🎯 **Smooth Animations** – Tween-based tile movements
- 🌍 **Localization Ready** – Supports multiple languages (EN/RU)
- 🏆 **Win Screen** – Celebrate your victory with stats and confetti

## 🚀 Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/mdimai666/godot-15.git
cd godot-15
```

### 2. Open in Godot

1. Launch **Godot Engine 4.6+**
2. Click **Import** and select the `project.godot` file
3. Wait for the .NET solution to be generated

## 🎮 How to Play

1. **Goal**: Arrange tiles in order from 1 to 15 (left to right, top to bottom)
2. **Controls**: Tap/click any tile adjacent to the empty space
3. **Win**: All tiles in correct order with empty space in bottom-right corner


```
Project → Export → [Your Platform] → Export Project
```
